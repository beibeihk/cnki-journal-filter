# Privacy Policy / 隐私政策

Last updated: 2026-05-14

## English

CNKI Journal Filter does not collect, transmit, share, sell, or store personal data on external servers.

The extension only stores the following data locally in the user's browser:

- selected preset journals
- user-entered custom journal names

This local storage is used only to restore the user's last filtering choices.

The extension does not:

- create user accounts
- require login
- use analytics or advertising SDKs
- send browsing data to the developer
- execute remote code

### Permissions Used

- `activeTab`: interact with the current CNKI tab
- `storage`: save the user's filter selections locally
- `scripting`: inject the content script if the CNKI page needs it
- CNKI host permissions: run only on CNKI-related domains declared in the manifest

### Support

For support or questions, please use the GitHub repository issues page for this project.

## 中文

知网期刊筛选器不会收集、上传、出售、共享或在外部服务器保存任何个人数据。

插件仅会在用户自己的浏览器本地保存以下信息：

- 已选择的预置期刊
- 用户手动输入的自定义期刊名称

这些本地数据仅用于恢复用户上一次的筛选配置。

本插件不会：

- 创建用户账号
- 要求登录
- 集成统计或广告 SDK
- 向开发者发送浏览数据
- 执行远程代码

### 权限说明

- `activeTab`：与当前的知网页签交互
- `storage`：在本地保存筛选配置
- `scripting`：在需要时向知网页面注入脚本
- 知网域名权限：仅在清单文件声明的 CNKI 相关域名下运行

### 支持方式

如需反馈问题或建议，请通过本项目的 GitHub Issues 页面联系。
